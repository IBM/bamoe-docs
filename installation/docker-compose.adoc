{CANVAS} and its related services can be run on a single machine using Docker Compose for {CANVAS}. 

== Prerequisites

You need Docker and Docker Compose to run the {CANVAS}.
See https://docs.docker.com/get-docker/[Docker Installation] and https://docs.docker.com/compose/install/[Compose Plugin Installation] for more information.

The `bamoe-{VERSION}-docker-compose.zip` can only be downloaded from IBM Passport Advantage, and you can find information on the https://www.ibm.com/support/pages/node/7151078[Support and Download Page].

== Run

To Run {CANVAS} with Docker Compose follow these steps:

. Extract the downloaded `bamoe-{VERSION}-docker-compose.zip`.
. In a terminal, navigate to the extracted folder: `bamoe-{VERSION}-canvas-docker-compose`.
. Run the command:
+
[source,shell]
----
$ docker compose up
----

{CANVAS} will be available at http://localhost:9090.
