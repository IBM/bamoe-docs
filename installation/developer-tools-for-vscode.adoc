= VS Code Extensions 
IBM; KIE Community
include::../partials/attributes.adoc[]

{PRODUCT_SHORT} provides some VS Code Extensions for developers to work with their Decision and Workflow projects.
These extensions are all available through the {PRODUCT_SHORT} Developer Tools bundle extension.

{PRODUCT_SHORT} Developer Tools bundles these extensions:

KIE BPMN Editor:: https://marketplace.visualstudio.com/items?itemName=kie-group.bpmn-vscode-extension

KIE DMN Editor:: https://marketplace.visualstudio.com/items?itemName=kie-group.dmn-vscode-extension

It is available at the Visual Studio Marketplace, at https://marketplace.visualstudio.com/items?itemName=IBM.bamoe-developer-tools.
Please refer to https://code.visualstudio.com/docs/editor/extension-marketplace[documentation] about installing extensions in VS Code.
